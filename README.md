# Vue 3 + RealWorld API+ X Layout

[Demo](https://x-realworld-vue-jyun.vercel.app/)

這是一個使用 Vue 3、RealWorld API 和 X (Twitter) layout 設計構建的 side project

## 實作項目

- RWD Layout
- 淺色/深色主題
- 多國語系
- 串接 API
- 使用者登入/註冊
- 首頁文章動態
- 文章新增/編輯/刪除
- 留言新增/刪除
- 喜歡文章
- 分享文章
- 用戶個人頁/貼文/喜歡的內容
- 追隨用戶
- 編輯個人資料
- 查詢文章
